# README – Website Gốm Sứ Artisan

## 1. Chủ đề dự án

Tên dự án: **Website bán gốm sứ thủ công – Gốm Artisan**

Website được xây dựng nhằm mô phỏng một cửa hàng bán gốm sứ online.  
Trang web giới thiệu các sản phẩm gốm như bình hoa, ly gốm, bộ ấm trà và các sản phẩm trang trí.

Mục tiêu của dự án:

- Thiết kế giao diện website bán hàng đơn giản
- Thực hành xây dựng layout bằng HTML và CSS
- Tạo bố cục website rõ ràng và dễ sử dụng
- Áp dụng các kỹ thuật như Flexbox và CSS Grid

Website gồm các phần chính:

- Header (Menu điều hướng)
- Hero section (Banner giới thiệu)
- Feature section (Lý do chọn sản phẩm)
- Sản phẩm nổi bật
- Banner quảng cáo
- Sản phẩm mới
- Newsletter đăng ký email
- Footer (Thông tin liên hệ)

## 3. Phân công công việc

### Thành viên 1
- Xây dựng cấu trúc HTML
- Tạo các section chính của website:
  - Header
  - Hero
  - Feature
  - Product
  - Footer

### Thành viên 2
- Thiết kế giao diện bằng CSS
- Sử dụng Flexbox để căn chỉnh header
- Sử dụng CSS Grid cho layout sản phẩm
- Tạo hiệu ứng hover cho các thẻ sản phẩm

### Thành viên 3
- Chuẩn bị nội dung sản phẩm
- Viết mô tả sản phẩm
- Thêm giá và hình ảnh minh họa

### Thành viên 4
- Kiểm tra lỗi giao diện
- Chỉnh sửa layout
- Hoàn thiện và đóng gói project
